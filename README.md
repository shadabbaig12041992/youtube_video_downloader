# Youtube Video Downloader
Paste any youtube video link and download that video

<h1>Requirements</h1>

    Python 3.8+
    

<h1>Installation</h1>

    Install Python 3.8+.
    pip install -r requirements.txt
    

<h1>Usage</h1>

    Clone this repository.
<h1>Run the program.</h1>
    
    Python video_downloader.py
